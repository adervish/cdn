What portion of traffic from www.amazon.com came from which CDN?

Save a session for www.amazon.com and this tool will figure out what % of the traffic came from which CDN. Watch the video to see how to save a session file.

The maximum size of a session file is 32mb, loading www.amazon.com with the cache clean seems to come it at half this, but if you browser around too much more, it is likely to get too big to process.

It's important to switch Chrome to incognito mode so you get a clean cache.

Different sites code the CDN responses differently, this MIGHT work on sites other than Amazon, but it probably won't!
